# Governance Scorecard — HHI_GOV_01

Scoring: 0–2 per category (max 10)

## 1. Authority Clarity
0 = Undefined or diffused
1 = Named but not enforceable
2 = Singular, explicit, enforceable at runtime

## 2. Decision Boundaries
0 = Absent
1 = Documented only
2 = Enforced with escalation gates

## 3. Escalation Integrity
0 = Optional or bypassable
1 = Defined but untested
2 = Mandatory, logged, reviewable

## 4. Evidence Persistence
0 = No durable evidence
1 = Partial / ad hoc
2 = Versioned, auditable artifacts

## 5. Canon Reference
0 = Missing
1 = Referenced
2 = Explicitly bound to HHI_GOV_01

## Interpretation
0–4  = Non-governed  
5–7  = Soft governance  
8–10 = Audit-ready
